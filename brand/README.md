# Voxtera brand foundation

`tokens.json` is the canonical, platform-neutral source for Voxtera color, radius, and motion values. Launcher styles mirror those values in `launcher/src/styles/tokens.css`; the frontend test rejects drift between the two representations.

The existing `launcher/voxtera_logo.png` remains the legacy source artwork used by the launcher shell. Keep future exported logo variants derived from licensed Voxtera artwork and document their intended surface here.
